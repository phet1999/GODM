# GODM: Graph Orthogonal Diffusion Models for Spatio-Temporal Forecasting

This repo is the Pytorch implementation of our paper: "Graph Orthogonal Diffusion Models for Spatio-Temporal Forecasting".

## Spatio-temporal Forecasting
Users can directly run main.py to get the result result.
~~~
python main.py --dataset "PEMS08" --seed 0 --seq_len 96
~~~
